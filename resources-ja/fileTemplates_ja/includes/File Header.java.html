これは、#parse ディレクティブを使用してファイル テンプレート (テンプレート タブ) にインクルードできるコード フラグメントを含むビルトイン テンプレートです。
このテンプレートは編集可能です。静的テキスト (コードとコメント) 以外に、マクロのように対応する値に展開されるあらかじめ定義されている変数を使用することもできます。
あらかじめ定義されている変数は以下の値を使用します :
${PACKAGE_NAME}   新規クラス/インターフェイスが作成されるパッケージの名前
${USER}   現在のユーザー (システム ログイン) 名
${DATE}   現在のシステム日付
${TIME}   現在のシステム時間
${YEAR}   現在の年
${MONTH}   現在の月
${DAY}   現在の月日
${HOUR}   現在の時間
${MINUTE}   現在の分
${PROJECT_NAME}   現在のプロジェクト名
