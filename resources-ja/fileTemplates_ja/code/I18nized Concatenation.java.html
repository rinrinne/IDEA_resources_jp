これは、国際化された Java コード式 (例えば、String s = MessageFormat.format(myResourceBundle.getString("hello {0}"), name);) で置換文字列結合 (例えば、String s = "Hello " + name;) に使用されるビルトイン テンプレートです。
このテンプレートは編集可能です。Java 式とコメント以外に、対応する値に展開されるあらかじめ定義されている変数を使用することもできます。
あらかじめ定義されている変数は以下の値を使用します :
${RESOURCE_BUNDLE}   型 java.util.ResourceBundle (このコンテキストで利用可能なインスタンス) の式。
${PROPERTY_KEY} ${PARAMETERS}   対応するプロパティ ファイルで指定されたプロパティ キーの名前。通常は、ResourceBundle.getString() メソッド パラメータに使用された値。
${PROPERTY_VALUE}   対応するプロパティ ファイルで指定されたプロパティの値。オリジナルの Java 文字列リテラル値。
