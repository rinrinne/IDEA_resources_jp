これは、例えば、抽象クラスの実装/抽象メソッドの実装アクションを使用して、または [コード] - [メソッドのインポート...] (Ctrl+I) 機能を呼び出して、プログラムが実装したメソッドのボディを生成するときに使用されるビルトイン テンプレートです。
このテンプレートは編集可能です。Java 式とコメント以外に、対応する値に展開されるあらかじめ定義されている変数を使用することもできます。
あらかじめ定義されている変数は以下の値を使用します :
${RETURN_TYPE}   作成したメソッドの戻り型
${DEFAULT_RETURN_VALUE}   メソッドのデフォルトの戻り値
${METHOD_NAME}   実装したメソッドの名前
${CLASS_NAME}   メソッドが実装されたクラスの修飾名
${SIMPLE_CLASS_NAME}   メソッドが継承されたクラスの非修飾名
All the predefined variables from "File Header" Include template are also available (e.g. ${USER}, ${DATE}, etc)
